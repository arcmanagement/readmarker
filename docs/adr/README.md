# Architecture Decision Records

Each ADR records why a significant design decision was made, so the axis doesn't
drift later. When a change request contradicts an ADR, point at the ADR and its
rationale rather than re-arguing from scratch.

Format per record: Context, Decision, Alternatives considered, Consequences.
